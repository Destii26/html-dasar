Pengertian HTML dan Sejarahnya

Hypertext Markup Language atau HTML adalah bahasa markup standar yang digunakan untuk membuat halaman website dan aplikasi web. Sejarah HTML diciptakan oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertama HTML dirilis oleh Tim Berners Lee pada tahun 1991, yang memiliki 18 tag. Sejak saat itu, setiap kali ada versi barunya, pasti akan selalu ada tag dan attribute (tag modifier) yang juga baru.

Berikut adalah beberapa dasar-dasar dalam HTML:

  • Tag: Tag adalah elemen dasar dalam HTML yang ditempatkan di dalam tanda < dan >. Contohnya: <p> untuk paragraf.
  • Elemen:Elemen adalah kombinasi dari tag pembuka dan penutup. Contohnya: <h1>Judul</h1> untuk judul.
  • Atribut: Atribut memberikan informasi tambahan pada tag. Contohnya: <a href="https://www.example.com">Tautan</a> untuk tautan.
  • Struktur Dasar: Setiap dokumen HTML memiliki <html>, <head>, dan <body>.
  • Heading Tags: <h1> sampai <h6> untuk judul dengan level yang berbeda.
  • Paragraph Tags: <p> untuk membuat paragraf teks.
  • Link Tags: <a> untuk tautan ke halaman atau sumber eksternal.
  • Image Tags: <img> untuk menyisipkan gambar.

Elemen HTML

<p> - Paragraf

Ini adalah contoh paragraf teks dalam elemen <p>. Paragraf digunakan untuk mengelompokkan teks menjadi unit yang berbeda.

<a> - Tautan

Ini adalah contoh tautan menggunakan elemen <a>. Tautan dapat mengarahkan pengguna ke halaman lain atau sumber eksternal. Contohnya: Tautan Contoh.

<img> - Gambar

Ini adalah contoh menyisipkan gambar menggunakan elemen <img>. Elemen ini membutuhkan atribut src (URL gambar) dan alt (teks alternatif).

<ul> dan <li> - Daftar Tak Berurutan

Ini adalah contoh daftar tak berurutan menggunakan elemen <ul> (unordered list) dan <li> (list item):
  • Item 1
  • Item 2
  • Item 3

<ol> dan <li> - Daftar Berurutan

Ini adalah contoh daftar berurutan menggunakan elemen <ol> (ordered list) dan <li> (list item):
  1. Langkah 1
  2. Langkah 2
  3. Langkah 3

Pengertian Tag HTML

Tag HTML (Hypertext Markup Language) adalah elemen dasar yang digunakan dalam bahasa markup HTML. Tag HTML digunakan untuk memberi tahu browser bagaimana cara menampilkan konten di halaman web. Setiap tag HTML dikelilingi oleh tanda < dan > yang dikenal sebagai tanda "kurung siku".

Tag HTML memiliki dua jenis utama: 1. Tag Pembuka dan Penutup

Beberapa tag HTML memiliki tag pembuka dan penutup yang berpasangan untuk membungkus konten di antara keduanya. Contoh:

<p>Ini adalah contoh teks.</p>

2. Tag Tunggal

Beberapa tag HTML tidak memerlukan tag penutup dan disebut tag tunggal. Contoh:

<br> (ganti baris)
    <img src="gambar.jpg" alt="Gambar"> (gambar)

Setiap tag juga dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. Contoh atribut: href dalam tag <a>.

Ketika bekerja dengan bahasa markup ini, Anda menggunakan struktur kode sederhana (tag dan attribute) untuk mark up halaman website.

Struktur HTML

Sebuah halaman HTML memiliki ekstensi file html. Dalam sebuah dokumen HTML terdiri dari blok-blok elemen HTML. Sebuah elemen HTML terdiri dari tag dan atribut HTML. Tag adalah individu dari struktur html, kita harus membuka dan menutup tag dengan garis miring dan tanda pembuka serta penutup tag. Ada beberapa variasi dengan tag beberapa di antaranya adalah tag self-closing yang tidak perlu ditutup dan ada juga tag kosong yang bisa kita tambahkan atribut apa saja di dalamnya. Atribut adalah properti tambahan dari tag html yang mendefinisikan properti dari setiap tag html seperti width, height, control, loop, input, dan autoplay. Atribut ini juga membantu menyimpan informasi dalam meta tag dengan menggunakan atribut name, content, dan type attributes.. secara lengkap dokumen html memiliki struktur sebagai berikut

Heading Tag dalam HTML

Heading tag dalam HTML digunakan untuk menampilkan judul atau tajuk pada halaman web. Tag-tag ini memiliki level yang berbeda, dimulai dari <h1> yang merupakan judul utama hingga <h6> yang merupakan judul dengan level terendah.

Contoh Penggunaan Heading Tag

Contoh penggunaan heading tag:

<h1>Ini adalah judul utama</h1>
<h2>Ini adalah subjudul</h2>
<h3>Ini adalah judul tingkat 3</h3>
Pada contoh di atas, <h1> digunakan untuk judul utama, <h2> untuk subjudul, dan seterusnya.

Paragraf Tag dalam HTML

Paragraf tag dalam HTML digunakan untuk mengelompokkan teks menjadi paragraf yang terpisah. Ini membantu dalam memisahkan dan mengatur konten teks dalam halaman web.

Contoh Penggunaan Paragraf Tag

Contoh penggunaan paragraf tag:

<p>Ini adalah contoh paragraf teks.</p>
<p>Ini adalah paragraf lain dengan teks yang berbeda.</p>

Pada contoh di atas, <p> digunakan untuk membuat paragraf teks yang berbeda.

Format Konten Teks

Paragraf tag juga digunakan untuk memformat konten teks seperti pemisahan baris (enter) dan ruang kosong. Namun, ini tidak mempengaruhi tampilan visual secara signifikan dalam halaman web.

Link Tag dalam HTML

Link tag dalam HTML (<a>) digunakan untuk membuat tautan atau hyperlink ke halaman web lain atau sumber eksternal. Ini memungkinkan pengguna untuk berpindah ke halaman lain dengan mengklik tautan tersebut.

Contoh Penggunaan Link Tag

Contoh penggunaan link tag:

<a href="https://www.example.com">Tautan Contoh</a>

Pada contoh di atas, href adalah atribut yang menentukan URL tujuan tautan.

Atribut Target

Atribut target digunakan untuk mengontrol di mana tautan akan terbuka. Contoh nilai atribut:

  • _blank: Tautan akan terbuka dalam jendela/tab baru.
  • _self: Tautan akan terbuka di jendela/tab saat ini (default).
<a href="https://www.example.com" target="_blank">Tautan Baru</a>

Image Tag dalam HTML

Image tag dalam HTML (<img>) digunakan untuk menyisipkan gambar ke dalam halaman web. Ini memungkinkan Anda menampilkan grafis, ikon, atau ilustrasi lainnya di halaman Anda.

Contoh Penggunaan Image Tag

Contoh penggunaan image tag:

<img src="gambar.jpg" alt="Gambar Contoh">

Pada contoh di atas, src adalah atribut yang menentukan URL gambar, dan alt adalah atribut yang memberikan teks alternatif jika gambar tidak dapat ditampilkan.

Atribut Lebar dan Tinggi

Anda dapat menggunakan atribut width dan height untuk mengatur dimensi gambar:

<img src="gambar.jpg" alt="Gambar Contoh" width="300" height="200">
